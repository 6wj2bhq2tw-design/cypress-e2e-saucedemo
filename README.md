# 🧪 Cypress E2E Automation for SauceDemo

## 📌 프로젝트 개요
이 프로젝트는 [SauceDemo](https://www.saucedemo.com/) 사이트를 대상으로
회원 로그인 및 상품 구매 흐름을 Cypress로 자동화한 E2E 테스트입니다.

## 🚀 주요 기능
- ✅ 로그인 성공/실패 시나리오
- ✅ 상품 장바구니 추가 및 결제 플로우
- ✅ GitHub Actions CI 연동
- ✅ 테스트 결과 자동 리포트

## 🧰 기술 스택
- Cypress
- Node.js 
- GitHub Actions
- VSCode

## 🧩 실행 방법
```bash
# 설치
npm install

# 로컬 테스트 실행
npx cypress run

# 브라우저 GUI 모드 실행
npx cypress open
